Shop Game was the first C++ project where I focused on understanding classes, structs, and functions through a small playable program. It is a console-driven market game that updates prices over time and lets the player buy and sell stock while managing risk and resources. Building it helped me learn how to organize code into reusable pieces and how object-like data structures support game state.
